# Soap Property Design Tool

A browser-based soap formulation calculator derived from the Excel workbook in this project.

## Features

- recipe calculator for up to three oils
- NaOH and KOH support
- color-coded soap property evaluation
- searchable oil reference table
- new oil submission flow with admin approval screens

## Files

- `index.html`: main tool
- `admin-tools.html`: admin approval screen
- `admin-password.html`: admin password management
- `soap-data.js`: embedded oil library for the browser
- `supabase/`: migrations and Edge Function source for approval workflow
